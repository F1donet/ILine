# ILine
Проект от компании «ILine»: система управления персоналом

1 Создана модель Flask-SQLAlchemy, отображающая структуру таблиц из базы данных, созданной в рамках проекта
«Онлайн-каталог сотрудников».  Модели включает все необходимые поля и связи между таблицами (например,
 связь «начальник-подчиненный»).  Связь между моделями реализуется через  внешние ключи.
2 Разработана страница, отображающая список всех сотрудников.  Каждый сотрудник представлен с полной
информацией (ФИО, должность, дата приема на работу, зарплата, начальник).
3 Реализована сортировка по заработной плате, используя параметры GET-запроса.
4 Реализован функционал поиска сотрудников по ФИО (полное или частичное совпадение).  Результаты поиска отображаются на той же странице.
5 Реализована возможность изменения начальника сотрудника путем прямого обновления записи в базе данных.
